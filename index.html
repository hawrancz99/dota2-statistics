Search player statistics by steam32 ID

You can find match ID in the match overview in the bottom left corner in game or you can log in into dotabuff.com and find it in your matches.
You need to use steam32 ID (9 digits). There are several ways how to get steam32 ID. These instructions are how to get for example your steam32 ID: 1.Go to store.steampowered.com and log into your account. 2.Go to your profile. 3.Copy the URL link. It will look like this, with numbers instead of the X: https://steamcommunity.com/profiles/765XXXXXXXXXXXXX 4.Go to steamid.xyz and paste the copied URL link into the search input and send it. 5.Now you should see your Steam32 ID. Copy it, go back to the Dota2 Statistics website and paste it into the player ID search input and hit "Search player".
